==== Get.exe spec memo ====


実行方法

	Get.exe [/E:01] [/H] 公開フォルダ定義ファイル

		/E:01
			簡易アップローダー用

		/H
			HEADリクエストとして動作
			ボディを返さない。

		公開フォルダ定義ファイル
			絶対パスまたは相対パス
			相対パスの場合、本プログラムが存在するディレクトリからの相対パスとなる。


公開フォルダ定義ファイル

	形式       ... テキスト
	文字コード ... CP932
	改行コード ... CR-LF 又は LF

	書式 = 1*(( HOST-NAME SP DOC-ROOT | COMMENT ) NEW-LINE ) EOF

		HOST-NAME = 1*( 0x21 〜 0x7e )
		SP = 0x20
		DOC-ROOT = 1*( パスに使用できる文字 )
		COMMENT = 0x3b *( 改行以外の文字 )
		NEW-LINE = *1( 0x0d ) 0x0a
		EOF = *1( 0x1a )

		各行の HOST-NAME と DOC-ROOT をホスト名と対応する公開フォルダの組み合わせとする。
		ホスト名は大文字・小文字を区別しない。
		ホスト名は重複してはならない。
		セミコロンで始まる行はコメントと見なす。

		公開フォルダ
			絶対パスまたは相対パス
			相対パスの場合、公開フォルダ定義ファイルが存在するディレクトリからの相対パスとなる。


ホスト名

	HTTPリクエストヘッダの Host フィールドの値から取得する。
	判定には大文字・小文字を区別しない。


とりあえずHTTPサーバーとして動かす手順

	1. このフォルダに Service.dat を以下の内容で作成する。

		GET .\Get.exe DocRoot.dat

	2. このフォルダに DocRoot.dat を以下の内容で作成する。

		default .

	3. このフォルダに index.html を適当な内容で作成する。
	4. 以下のコマンドを実行する。

		HTT.exe /S Service.dat

	5. ブラウザから http://localhost/ にアクセスして index.html の内容が表示されたら成功
